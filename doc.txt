activer l'environement avant de commencer l'analyse le nom de l'environnement RNA_Artefact
j'ai installé snakemake sur cette env 
commande pour installer les packages R:
conda install r-name_of_package
commande : conda install -c bioconda -c conda-forge snakemake
mettre les packages utilisées dans RNA_artefcat dans un fichier requirements.txt
commande: conda list -e > requirements.txt
creer à partir de ce fichier l'environement 
conda create --name name_env --file requirements.txt
commande pour lancer snakemake pour les artefacts RNA avec parametre config
snakemake -s snakefile --configfile=config.yaml --config f="hajer.txt" --forceall -c1
commade pour lancer snakemake sur cluster
snakemake -F -s snakefile --configfile config.yaml --cluster 'sbatch  --mem=1g -c 20'
